UMAP of modern and ancient humans.

An Interactive Explorer of population genetics study data.

Interact with the widgets on the left to query a subset of individuals to plot. Hover over the circles to see more information about each individual.

Original compiled genotype data taken from: AADR dataset.

For more detailed information on how the data was processed please see: https://github.com/thirtysix/population_genetics_AADR)
